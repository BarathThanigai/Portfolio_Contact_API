# Portfolio Contact API

A lightweight FastAPI backend that powers the contact form for my personal portfolio website. The API validates contact requests, securely sends emails via SMTP, and enables visitors to reach me directly from the portfolio without exposing sensitive credentials.

## Features

* Contact form API built with FastAPI
* Input validation using Pydantic
* Secure email delivery via SMTP
* Environment variable configuration using `.env`
* CORS support for frontend integration
* Clean and lightweight REST API
* Ready for deployment on platforms such as Render

## Tech Stack

* Python
* FastAPI
* Pydantic
* SMTP
* Python Dotenv
* Uvicorn
